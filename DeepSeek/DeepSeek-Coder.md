
# DeepSeek-Coder-7b-base-v1.5

* mô hình ngôn ngữ chuyên biệt cho lập trình (Code LLM)
* được thiết kế để "hiểu" và "viết" code một cách thông minh, tối ưu. 
* đọc hiểu toàn bộ một file code dài hoặc hiểu mối liên hệ giữa nhiều file trong một project nhỏ.
* Cơ chế Infilling: Hỗ trợ tính năng "điền vào chỗ trống" (FIM - Fill-In-the-Middle), cực kỳ hữu ích cho việc tự động hoàn thiện code trong các IDE.
* Thông minh hơn khi viết các hàm phức tạp, đưa ra các giải pháp hiện đại và ít mã rác hơn như __CodeLlama-34B, StarCoder2__
* Tư duy Logic & Toán học: Khác với các model chỉ học thuộc lòng cú pháp, DeepSeek-Coder được tinh chỉnh để hiểu logic thuật toán, giúp việc gỡ lỗi (debug) và tối ưu hóa code chính xác hơn.
* Khả năng giải thích code khá chi tiết bằng tiếng Việt/Anh.
* Convert code từ ngôn ngữ này sang ngôn ngữ khác (ví dụ: Python sang Rust).
